# Auto BGM Personal

A **personal-use** automation tool for generating long-form ambient background music videos and uploading them to my own YouTube channel.

## What It Does

1. Combines AI-generated music tracks (created by me using Suno) into multi-hour BGM compilations.
2. Generates a thumbnail and metadata (title, description, tags) from local templates.
3. Uploads the resulting video to **my own YouTube channel** using YouTube Data API v3.

## Who Uses It

This app is used **only by me, the developer**. It is not offered to any other user, organization, or third party. No external accounts are connected.

## Permissions Used

- `https://www.googleapis.com/auth/youtube` — required to upload videos and set thumbnails on **my own channel**.
